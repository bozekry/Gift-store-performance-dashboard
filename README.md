# 🎁 Gift Shop Performance Dashboard
## 📖 Overview

This project presents an interactive Excel dashboard designed to analyze the sales performance of a gift shop. The dashboard transforms raw transactional data into meaningful business insights using Microsoft Excel, Power Query, Power Pivot, DAX, Pivot Tables, and Pivot Charts.

The project enables stakeholders to monitor key business metrics, identify sales trends, evaluate product performance, analyze customer purchasing behavior, and support data-driven decision-making.

## 🎯 Project Objectives

The primary objective of this project is to analyze gift shop sales data and provide interactive business insights that help stakeholders make informed decisions.

The dashboard focuses on:

- Monitoring sales performance
- Identifying top-selling products
- Evaluating revenue across occasions
- Tracking delivery performance
- Understanding customer purchasing behavior
- Supporting strategic business decisions
## 📂 Dataset

The project is built using three related datasets:

### Orders
Contains transaction details including:

- Order ID
- Customer ID
- Product ID
- Quantity
- Order Date
- Order Time
- Delivery Date
- Delivery Time
- Location
- Occasion

### Customers

Contains customer information such as:

- Customer ID
- Customer Name
- Gender
- City

### Products

Contains product information including:

- Product ID
- Product Name
- Category
- Price
